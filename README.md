Project Planning & Skill Management Tool (Vue.js)

A web-based application developed as part of the Software Project Seminar at the University of Bayreuth.
The tool supports efficient project planning, team skill assessment, and visual project structuring.

The application is deployed via GitHub Pages:
https://oliverschwall03-art.github.io/Software-Projektseminar/

Features:
- Dashboard overview for active projects
- Project list and detail view
- Task planning with timeline visualization (Gantt-like)
- Import / export of planning data
- Ambidexterity & skill assessment tools

This project is built using:
- Vue 3
- Vite
- Vue Router
- JavaScript / HTML / CSS
- GitHub Pages (gh-pages branch for deployment)
